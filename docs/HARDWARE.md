# Hardware de referência

- Orange Pi Zero 3
- Webcam USB
- Servo 9g (pan horizontal)
- Fonte separada para servo

## Boas práticas

- Servo pode demandar corrente de pico; evite alimentar direto pelo SBC.
- Use GND comum entre Orange Pi e fonte do servo.
- Mantenha cabos curtos e aterramento adequado para reduzir ruído.
